# AMOS Content Execution Plan

## Главная цель

Создать автоматизированную контентную систему для развития:

- личного бренда Дениса
- AutoVector
- будущих проектов внутри AMOS

---

# Основная стратегия

Денис является главным брендом.

AutoVector является коммерческой платформой.

AMOS является операционной системой бизнеса.

---

# Цели на первый год

1. Создать узнаваемый бренд Дениса.
2. Собрать аудиторию вокруг AutoVector.
3. Получать лиды через контент.
4. Автоматизировать создание контента.
5. Построить медиа-актив с собственной аудиторией.
