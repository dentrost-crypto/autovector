## Operational Mode: Trend Scout

Trend Scout mode собирает рыночные референсы, примеры постов, темы конкурентов и идеи, которые могут вдохновить Content Creator Agent.

Этот режим не копирует конкурентов и не сохраняет материалы как готовый контент для публикации. Он сохраняет только наблюдения:

- structure;
- topic;
- hook;
- CTA;
- whyItWorks.

Собранные элементы используются как сырьё для создания оригинального контента AutoVector.

Правила режима:

- не копировать чужие тексты;
- не переносить чужие обещания, цифры или кейсы без проверки;
- выделять только идею, механику подачи и причину, почему материал может работать;
- адаптировать выводы под голос AutoVector: спокойствие, доверие, прозрачность, сопровождение сделки;
- передавать Content Creator Agent только безопасные вдохновения, а не готовые публикации.

Первый рабочий скрипт режима:

```bash
npm run trend:scout
```

Данные режима:

- `AMOS/intelligence/sources.json`
- `AMOS/intelligence/reference_library.json`
