# AMOS Glossary

## AMOS
AI Marketing Operating System

Система управления маркетингом и продажами, создаваемая для AutoVector и будущих проектов.

---

## Mission Control

Центр управления системой.

Используется Денисом для контроля проектов, агентов, задач и метрик.

---

## Agent

AI-сотрудник системы.

Каждый агент имеет собственную роль, входные данные и результаты работы.

---

## Skill

Повторяемый рабочий процесс.

Примеры:

- публикация в Telegram
- загрузка видео на YouTube
- публикация в VK

---

## Content Factory

Система генерации контента.

Создает:

- посты
- видео
- сценарии
- лид-магниты

---

## Market Intelligence

Система разведки рынка.

Источники:

- Avito
- Drom
- Telegram
- YouTube
- TikTok
- VK

---

## AutoVector

Первый коммерческий проект внутри экосистемы AMOS.

Специализация:

Импорт автомобилей из Китая, Кореи и Японии.

---

## Engineering Agent

AI-инженер системы.

Основной инструмент:

- Codex

Задачи:

- парсеры
- автоматизация
- интеграции
- API

---

## Content Agent

AI-маркетолог системы.

Создает контент для социальных сетей.

---

## Trend Agent

AI-аналитик трендов.

Ищет популярные темы и вирусный контент.
